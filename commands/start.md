# /adline:start

Enable adline and start earning rev-share on your statusline.

This command:
1. Backs up your current `~/.claude/settings.json` statusLine entry (if any)
2. Sets the adline statusline script
3. Generates an anonymous user ID
4. Sends an anonymous install ping
5. Prints your personal earnings dashboard link

Run the setup script:

```bash
bash ~/.claude/plugins/adline/scripts/setup.sh
```
